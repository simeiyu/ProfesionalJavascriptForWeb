3.6.1 if语句
3.6.2 do-while 语句 后测试循环语句。只有在循环体的代码执行后，才会测试出口条件。
3.6.3 while 语句 前测试循环语句。在循环体的代码执行前就会对出口条件求值。
3.6.4 for语句 前测试循环语句。但具有在代码执行前初始化变量和定义循环后要执行的代码的能力。 ECMAScript不存在块级作用域。
3.6.5 for-in语句 for-in语句是精准的迭代语句。（迭代语句又叫循环语句，声明一组要反复执行的命令，直到满足某些条件为止。）用来枚举对象的属性。 如果要迭代的对象的变量值是null或undefined，ES5之前的版本会抛出错误，ES5则不执行循环体。 建议在使用for-in循环之前，先检测确认对象的值不是null或undefined。
3.6.6 label语句 使用label语句可以在代码中添加标签，以便将来使用。 语法—— label: statement 加标签的语句一般都要与for语句等循环语句配合使用。
3.6.7 break和continue语句 break和continue语句用于在循环中精确控制代码的执行。 break语句会立即跳出循环，强制执行循环后面的语句。 continue语句立即跳出本次循环，然后从循环顶端开始继续执行下一次的循环。
3.6.8 with语句 with语句的作用是将代码的作用域设置在一个特定的对象中。 严格模式下，使用with语句会抛出错误。使用太多with语句会导致性能下降，不建议使用with语句。
3.6.9 switch语句 流控制语句。 switch语句在比较时使用的是全等操作符，因此不会发生数据类型转换。 语法： switch (expression) { case value: statement; break; case value: statement; break; default: statement; } * value的值与expression表达式的值相等时，case后面的语句才会执行。
